临床陪护介绍

护工需经过相关专业知识和技能的培训后方能上岗，承担病区内非技术性的护理工作。

协助病人的起居、洗漱、仪容仪表的整理。不能自理的患者帮其完成如：洗脸、梳头、口腔清洁、假牙护理、擦身、更衣，协助如厕或使用便盆、便壶等工作。

协助病人满足饮食的营养需求如：喂饭、喂水，整理餐后用具。

协助病人上下床，坐轮椅。

协助患者保持床单位干净整洁，卧位正确舒适，负责整理病室环境，包括床上、床下物品。

协助医护观察病情如：病人发热，液体突减慢，局部肿胀等。收送各标本。

保持患者的舒适并保证安全（重症、昏迷、老年患者使用床档的管理，保持病人的卧位正确、舒适，但对术后重症、牵引的患者应在护理人员指导下移动病人）。

完成晨、晚间护理。做好护理记录（如：出入量）工作，保持病房内干净整洁、无异味。
